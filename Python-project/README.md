SMART EXPENSE TRACKER

The Smart Expense Tracker is a Python-based CLI application that helps users record, categorize, and analyze their daily expenses. It provides meaningful insights into spending patterns and helps users make better financial decisions.

Objectives: Record daily expenses (date, category, amount, description) Categorize spending (Food, Travel, Bills, Others, etc.) Analyze expenses and generate useful insights Provide monthly summaries

Features:

CLI-based interactive system (menu-driven)
Add and view expenses
Monthly expense summary
Category-wise expense breakdown
Pie chart visualization using Matplotlib
Detect highest spending category
Smart insights and suggestions
Technologies Used: --> Programming Language Python 3 --> Libraries Used csv → For storing and reading expense data matplotlib → For data visualization (pie chart) os → For file handling and checking file existence
