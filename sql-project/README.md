THE-DIGITAL-LIBRARY-AUDIT

A simple Digital Library Management system using MySQL to track book issues, identify overdue returns, generate penalty reports, and analyze popular book categories.

#Objectives The goal of this project is to:

Maintain records of books and students
Track issued and returned books
Identify overdue books (not returned within 14 days)
Generate a penalty report
Analyze most borrowed book categories
Identify inactive students
#Sql concepts used

Joins (INNER JOIN, LEFT JOIN)
Aggregate Functions (COUNT, MAX)
GROUP BY and HAVING
Subqueries
Date Functions (CURDATE, DATEDIFF, INTERVAL)
